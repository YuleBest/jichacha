# 机查查 - 手机代号查询工具

一个基于 Vue 3 + Vuetify 3 开发的设备型号查询工具，支持按品牌和关键词搜索设备型号信息

## 功能特性

- **全局搜索**: 支持按设备名称、代号、型号等关键词搜索
- **品牌分类**: 按品牌浏览设备信息
- **一键复制**: 快速复制设备代号和型号
- **设备类型筛选**: 支持手机和平板分类筛选

## 技术栈

- **前端框架**: Vue 3
- **UI 框架**: Vuetify 3
- **数据处理**: PapaParse
